# Setup Instructions

## Prerequisites
- Python 3.11
- Docker
- Docker Compose

## Installation
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies:
   ```bash
   1 pip install -r requirements.txt
   ```

4. Set up the database and environment variables as needed.
5. Run the application:
   ```bash
   1 docker-compose up
   ```
